# Bano Qabil Project Website

A modern, responsive web application designed for **Bano Qabil**, showcasing its initiatives, courses, and student empowerment programs. This project serves as a comprehensive digital portal to help users explore available technical tracks, register for courses, and stay updated with the community.


## ✨ Features

- **Responsive Design:** Optimized for all screen sizes, including smartphones, tablets, and desktops.
- **Course Catalog:** Clear breakdown of available technical tracks (e.g., Web Development, Cybersecurity, Digital Marketing).
- **Student Registration Portal:** An interactive and intuitive signup/inquiry form for prospective students.
- **About & Mission Section:** Insights into Bano Qabil's vision of empowering youth through accessible education.
- **Dynamic UI Elements:** Clean navigation, modern grid layouts, and interactive components built with vanilla JavaScript.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Styling/Layout:** Flexbox, CSS Grid
- **Version Control:** Git & GitHub
